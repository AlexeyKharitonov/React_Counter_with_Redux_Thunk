# Для запуска проекта, необходимо выполнить следующие шаги:

1. Склонировать репозиторий проекта по ссылке https://github.com/AlexeyKharitonov/React_Counter_with_Redux_Thunk на свой компьютер.

```
git clone https://github.com/AlexeyKharitonov/React_Counter_with_Redux_Thunk.git
```

2. Открыть терминал (или командную строку) и перейти в корневую директорию проекта.

```
cd React_Counter_with_Redux_Thunk
```

3. Установить все необходимые зависимости:

```
npm install
```

4. Запустить проект:

```
npm run start
```
